5. User Interface Design


5.1 Overview of User Interface


In this world of technology, in order to survive in the market a piece of product needs to be user friendly. User interface is the front end of a software platform which interacts with the user. Keeping this in mind, our interface should be in such a way that it gives the user a proper insight of our product. Our UI will be text based, graphical or audio based depending on the purpose.
